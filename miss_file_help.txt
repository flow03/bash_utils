Create list of numbered files in directory and find missing numbers

miss_file [directory_path] [flags]
miss_file [flags] [directory_path] 

-p		manually specify directory
-b		manually specify end
-e		manually specify extension, default work with jpg and png
-l		column output
-q		disable questions
-d		also count duplicate numbers, may use with -l
-w		don't close console for windows after
-h		show this help, discard other flags
